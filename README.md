# expand-utils [![NPM version](https://badge.fury.io/js/expand-utils.svg)](http://badge.fury.io/js/expand-utils)  [![Build Status](https://travis-ci.org/jonschlinkert/expand-utils.svg)](https://travis-ci.org/jonschlinkert/expand-utils)

> Utils shared by the expand libs.

## Install

Install with [npm](https://www.npmjs.com/)

```sh
$ npm i expand-utils --save
```

## Usage

```js
var util = require('expand-utils');
```
