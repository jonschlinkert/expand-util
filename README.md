# expand-utils [![NPM version](https://img.shields.io/npm/v/expand-utils.svg)](https://www.npmjs.com/package/expand-utils) [![Build Status](https://img.shields.io/travis/jonschlinkert/expand-utils.svg)](https://travis-ci.org/jonschlinkert/expand-utils)

> Utils shared by the expand libs.

## Install

Install with [npm](https://www.npmjs.com/):

```sh
$ npm i expand-utils --save
```

## Usage

```js
var util = require('expand-utils');
```
